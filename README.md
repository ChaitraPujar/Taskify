# Taskify

Taskify is a simple task management application built as a technical assignment.

## Tech Stack
- ASP.NET Core Web API (.NET 8)
- Entity Framework Core
- SQL Server LocalDB
- JWT Authentication
- Angular (frontend)

## Architecture
The solution follows a layered architecture:

- **Taskify.API** – HTTP API layer
- **Taskify.Core** – Domain models and contracts
- **Taskify.Infrastructure** – Data access and service implementations

- Solution and project structure created
